人与 Agent 的社会契约

AI 时代，人和 agent 之间需要新的社会契约。

草稿脚手架 观点型博客

问题框架

这篇文章从 productivity 转向 legitimacy。如果 agent 变成消息系统、工作场景和消费产品中的长期协作者，旧的默认信任假设就不够了。文章会追问：哪些权利、义务和监督机制应该治理人和 agent 的关系？

初始观点

  • 把任务委托给 agent 时，必须同时给出明确边界和清晰的撤销权。
  • 对 agent 的信任不只取决于能力，也取决于可控性、可恢复性和可审计性。
  • 未来的社会契约会同时涉及人、agent，以及居中调度它们的平台。

草稿大纲

  1. 为什么只有能力还不够
  2. 委托、权限与撤销
  3. 平台对用户负有什么义务
  4. 为什么 trustworthiness 比 benchmark theater 更重要
  5. 走向一种实用的人-agent 社会契约
