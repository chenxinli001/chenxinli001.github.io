Blog No. 2

Intent Over Information

AI 时代，稀缺的不是信息，而是可执行的意图表达。

草稿脚手架 观点型博客

问题框架

这篇文章探索一个想法：AI 会把信息论推向更主观、更面向行动的框架。在有 agent 的系统里，关键传递的往往不是一包事实，而是一组可解释的目标、约束、偏好和成功标准。

初始观点

  • 没有 agent 时，信息传递主要服务于人的理解。
  • 有 agent 时，沟通越来越服务于协同执行。
  • 稀缺资源变成了结构良好的意图：它要能在被翻译成行动时保持稳定。

草稿大纲

  1. 从客观 bits 到主观 relevance
  2. 意图和普通信息有什么不同
  3. 为什么 agent 压缩了从语言到行动的距离
  4. “好的意图表达”在实践中长什么样
  5. 这如何改变界面、workflow 和教育
